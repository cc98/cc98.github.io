wget的下载与安装：  
下载地址：[ftp://ftp.cs.cuhk.edu.hk/pub/gnu/gnu/wget](ftp://ftp.cs.cuhk.edu.hk/pub/gnu/gnu/wget)  
安装：  
先把wget下载到的tar文件解压，然后cd到wget目录下

```bash
./configure make make install make clean
```

简单使用：wget url(要下载的地址)
